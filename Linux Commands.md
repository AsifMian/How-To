 Important Linux Commands 
</h1>

 
 ## 1.Opening a file as administrator 
  
  ```sh
    sudo nano fileName
    let say we want to open .env file
    sudo nano .env
  ``` 
    
2. **Moving to directories**
  
  ```sh
   # Move to a directory
    cd directoryName
   # Move one step above (to previous directory)
     cd ../
   # Move to root
    cd 
 ```
